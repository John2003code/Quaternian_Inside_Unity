# Quaternian_Inside_Unity
A project exploring 3D math and surface-aligned character movement in Unity.

## Features
* **Custom Quaternion Library:** Implemented Hamiltonian products, Matrix-to-Quaternion conversion, and RotateVector from scratch.
* **Surface Alignment:** A player controller that aligns to any surface normal (e.g., walking on a sphere).
* **Work in Progress:** Currently debugging coordinate space inversions in the custom movement logic.
* It has bugs, in future I will try to fix that.
